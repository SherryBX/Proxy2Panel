# DESIGN.md

## Theme
深色主题，接近煤黑，不用纯黑。

## Colors
- Background: oklch(0.15 0.01 220)
- Panel: oklch(0.19 0.012 220)
- Panel Alt: oklch(0.23 0.014 220)
- Text: oklch(0.9 0.01 220)
- Muted: oklch(0.72 0.01 220)
- Accent: oklch(0.72 0.13 190)
- Success: oklch(0.74 0.14 155)
- Warning: oklch(0.77 0.12 85)
- Danger: oklch(0.67 0.17 25)

## Typography
- UI Font: Inter, Segoe UI, system-ui, sans-serif
- Mono: JetBrains Mono, Consolas, monospace

## Layout
- 左侧单栏导航
- 顶部状态条
- 主内容区域以模块分区，不用营销式容器
- 图表区域优先宽版，不堆小卡片

## Components
- 细边框面板
- 扁平按钮，少量强调色
- 紧凑表格
- 日志视图使用等宽字体
